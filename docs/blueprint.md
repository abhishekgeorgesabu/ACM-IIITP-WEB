# **App Name**: ACM IIIT Pune

## Core Features:

- Homepage with About Us: Landing page with an introduction to the ACM student chapter.
- Events Calendar: Display upcoming events with details and registration links.
- Membership Information: Details on how to become a member, benefits, and levels.
- Team Profiles: Profiles of the student chapter's team members with bios and pictures, revealed on hover. With faculty advisor on top, 3 Office Bearers, 6 vertical heads and members
- Contact Form: A contact form for inquiries and communication.

## Style Guidelines:

- Dark scheme: Background in dark gray (#333333) to give a professional feel.
- Primary color: Blue (#3498DB) to ensure sufficient contrast, provide visual interest, and match professional theme.
- Accent color: Orange (#E67E22) for CTAs and interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text to ensure readability.
- Use a consistent set of line icons from a single library. Maintain a visual balance, by limiting filled icons to situations where additional emphasis is required.
- Responsive grid layout with clear content sections for easy navigation.
- Subtle hover animations for team member profiles and smooth transitions between page sections.
